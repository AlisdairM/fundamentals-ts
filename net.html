The <experimental/net> header is available if uint8_t, uint16_t, uint32_t, and uint64_t are provided by <cstdint>.

For each unsigned integer type unsigned-integral, there shall be explicit specializations of the hton() and ntoh() templates.

Byte order conversion

Network byte order is big-endian, or most signiﬁcant byte ﬁrst (RFC 2781 section 3.1). This byte order is used by certain network data formats as it passes through the network. Host byte order is the endianness of the host machine.
